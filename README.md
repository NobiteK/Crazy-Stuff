# 🎮 AutoHotkey Scripts Collection

All scripts created and maintained by me for personal use and learning purposes.

**⚠️ Disclaimer:** These scripts are for educational purposes. Use responsibly and in accordance with game/software terms of service.

## 🖱️ AutoClicker

High-performance auto-clicker with GUI, optimized timer precision, customizable speed, and random delay option. Supports left, right, and middle mouse buttons.

### Controls
| Key | Action |
|-----|--------|
| `F` | Toggle ON/OFF (customizable) |
| `Insert` | Alternative toggle |
| `End` | Close script |

## 🎯 TriggerBot

Pixel-based triggerbot that automatically shoots when detecting color changes at crosshair position. 80-200ms random shot delay, 15px color threshold.

### Controls
| Key | Action |
|-----|--------|
| `LAlt` | Hold to activate |
| `PgUp` | Toggle script ON/OFF |
| `End` | Close script |

## 🚪 DoorSpam

Made for CS2 - rapidly spams the 'E' key while holding mouse button 4 (side button) for spam opening doors.

### Controls
| Key | Action |
|-----|--------|
| `\` | Toggle script ON/OFF |
| `Mouse 4` | Hold to spam |
| `Home` | Close script |

| `PgUp` | Toggle ON/OFF |

## 💬 AutoMessage

Automatically sends a predefined message at regular intervals. Useful for chat automation.

### Controls
| Key | Action |
|-----|--------|
| `=` or `F1` | Toggle ON/OFF |
| `End` | Close script |

### Configuration
- `TEXT_TO_PASTE` - Text to be auto-pasted
- `INTERVAL` - Delay between messages
- `PRESS_ENTER` - Whether to press Enter after pasting
